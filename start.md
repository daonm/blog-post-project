# [개발 명세서] 올인원 블로깅 프로 (All-in-One Blogging Pro) 자동 포스팅 시스템

## 1. 프로젝트 개요
- **목표**: 아이디어 발굴부터 키워드 분석, 고품질 글쓰기, 이미지 생성, 소셜 홍보까지 블로그 운영 전 과정을 자동화/효율화하는 통합 솔루션 구축.
- **주요 타겟**: 티스토리, 워드프레스, 네이버 블로그 운영자.
- **핵심 기술**: Gemini API (AI Studio), Naver Search API (SEO), DALL-E (이미지), HTML/JS (인터랙티브 요소).

---

## 2. 주요 기능 및 모듈 설계

### 2.1 주제 아이디어 엔진 (Topic Generator)
사용자의 고민을 해결하기 위한 5가지 전문 분석 모델 제공:
1. **카테고리별 추천**: IT, 건강, 재테크, 라이프스타일 등 선택한 분야의 최신 트렌드 기반 주제 제안.
2. **E-EAT 기반 추천**: 구글 검색 엔진 최적화를 위해 경험(E), 전문성(E), 권위성(A), 신뢰성(T)을 충족하는 심층 주제 제안.
3. **에버그린 콘텐츠**: 시간이 지나도 가치가 변하지 않는 스테디셀러 주제 (예: 역사, 원리, 기초 가이드).
4. **롱테일 키워드 주제**: 검색량은 적지만 경쟁이 낮고 검색 의도가 명확한 세부 주제 (실시간 구글 검색 데이터 활용).
5. **메모/파일 기반**: 사용자의 메모, PDF, 강의 노트, Perplexity 검색 결과 등을 분석하여 블로그 글로 재구성.

### 2.2 키워드 파이터 (Keyword Analyzer)
- **네이버 API 연동**: 네이버 개발자 센터 API(Client ID, Secret)를 사용하여 키워드 경쟁도 분석.
- **SEO 전략**: 경쟁 블로그를 분석하고 상위 노출을 위한 최적의 전략 수립.
- **연동성**: 분석된 키워드와 주제를 클릭 한 번으로 '포스트 생성' 모듈로 전달.

### 2.3 포스트 생성 엔진 (Post Creator)
- **UI 구성**: 기사 제목(주제), 추가 요청 사항, 컬러 테마(7종) 설정.
- **이미지 옵션**: 
    - 대표 이미지 생성 유무 선택.
    - 본문 서브 이미지(최대 3개) 생성 유무 선택.
- **인터랙티브 요소**: 글의 몰입도를 높이는 '진단기' 또는 '계산기' 기능 삽입 (체크박스로 활성화).
- **인간적인 글쓰기 (AI 감지 회피)**:
    - **유형 A (친근형)**: 감정 표현, 개인적 경험, 대화체 사용.
    - **유형 B (전문가형)**: 논리적 구조, 고급 어휘, 신뢰감 있는 문체.
- **결과물 출력**: HTML 코드 형태 제공 및 추천 태그/키워드 자동 생성.

### 2.4 디자인 및 미디어 모듈 (Design & Media)
- **DALL-E 연동**: 고품질 블로그 이미지 실시간 생성.
- **썸네일 제작기 (Thumbnail Maker)**:
    - 생성된 이미지 위에 텍스트 오버레이.
    - 기능: 줄바꿈(슬러시 `/` 활용), 글꼴, 색상, 크기, 외곽선 설정.
    - 비율 선택 및 썸네일 전용 이미지 합성.
- **이미지 관리**: SEO 성능(로딩 속도)을 위해 Base64 인코딩이 아닌 개별 이미지 경로 방식으로 처리.

### 2.5 소셜 미디어 홍보 (SNS Marketing)
- 블로그 글 생성 후 자동으로 플랫폼별 최적화된 문구 생성:
    - **Threads/X**: 짧고 임팩트 있는 문구.
    - **Instagram**: 감성적인 표현 및 해시태그.
    - **Facebook**: 요약 및 공유 유도 문구.

## 3. 기술적 구현 디테일 (Logic & Workflow)

### 3.1 이미지 및 SEO 로직
- **Base64 방지**: 이미지 데이터를 HTML에 직접 포함하면 파일이 너무 커져 SEO 순위가 하락하므로, 이미지는 별도 다운로드/업로드 방식을 권장함.
- **Alt 태그 자동 생성**: 생성된 모든 이미지에 대해 검색 엔진이 인식할 수 있는 Alt 텍스트 자동 삽입.

### 3.2 인터랙티브 코드 핸들링 (티스토리 특화)
- 티스토리 에디터의 특성(기본 모드 전환 시 JS 소실)을 고려한 설계:
    - 코드 시작과 끝에 주석 주입: ``, ``.
    - 사용자가 HTML 모드에서 해당 구간의 코드를 쉽게 교체할 수 있도록 별도의 '인터랙티브 코드 복사' 버튼 제공.

### 3.3 프롬프트 엔지니어링 (AI 지침)
- **구조화**: 단순 메모 형태가 아닌 AI 스튜디오의 '앱 빌더' 형태를 활용하여 구조화된 명령 수행.
- **강제화**: 글자 수 부족 방지 및 인용 코드 삽입 등 정교한 지침 강제 적용.

## 4. 개발 단계별 로드맵
1. **1단계**: Gemini API 기반 5개 주제 추천 엔진 구축.
2. **2단계**: 네이버 검색 API 연동 키워드 분석 모듈 개발.
3. **3단계**: 본문 HTML 생성 로직 및 인간적인 글쓰기 스타일 적용.
4. **4단계**: DALL-E 이미지 생성 및 썸네일 텍스트 합성 엔진 개발.
5. **5단계**: 인터랙티브 요소(JS) 및 소셜 홍보 문구 생성기 연동.
