# 올인원 블로깅 프로 (All-in-One Blogging Pro)

블로그 운영의 모든 과정을 자동화하는 통합 솔루션입니다. 아이디어 발굴부터 키워드 분석, 고품질 글쓰기, 이미지 생성, 소셜 홍보까지 한 곳에서 처리할 수 있습니다.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 주요 기능

### 1. 주제 아이디어 엔진 🎯
5가지 전문 분석 모델로 완벽한 블로그 주제를 찾아보세요:
- **카테고리별 추천**: IT, 건강, 재테크 등 분야별 최신 트렌드 주제
- **E-EAT 기반 추천**: 구글 SEO 최적화를 위한 전문성 있는 주제
- **에버그린 콘텐츠**: 시간이 지나도 가치가 변하지 않는 주제
- **롱테일 키워드**: 경쟁이 낮고 검색 의도가 명확한 세부 주제
- **메모/파일 기반**: 메모, PDF, 강의 노트를 블로그 글로 재구성

### 2. 키워드 파이터 🔍
- 네이버 검색 API 기반 키워드 경쟁도 분석
- 월간 검색량 추정
- SEO 전략 자동 제안
- 연관 키워드 추천

### 3. 포스트 생성 엔진 ✍️
- AI 기반 고품질 블로그 글 자동 작성
- 2가지 글쓰기 스타일 (친근형/전문가형)
- 7가지 컬러 테마 선택
- 대표 이미지 및 본문 이미지 생성 옵션
- 인터랙티브 요소 (진단기/계산기) 추가
- HTML 코드 자동 생성
- 추천 태그 자동 생성

### 4. 디자인 및 미디어 🎨
- **이미지 생성**: DALL-E 3 기반 고품질 이미지 생성
- **썸네일 제작기**: 
  - 텍스트 오버레이
  - 줄바꿈 기능 (슬러시 `/` 활용)
  - 글꼴, 색상, 크기, 외곽선 커스터마이징
  - 다양한 비율 지원 (16:9, 1:1, 4:3)

### 5. 소셜 미디어 홍보 📱
플랫폼별 최적화된 홍보 문구 자동 생성:
- **Threads/X**: 짧고 임팩트 있는 문구
- **Instagram**: 감성적 표현과 해시태그
- **Facebook**: 요약 및 공유 유도 문구

## 🚀 시작하기

### 필수 요구사항
- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
- API 키:
  - [Google Gemini API](https://makersuite.google.com/app/apikey)
  - [OpenAI API](https://platform.openai.com/api-keys) (DALL-E 3)
  - [네이버 개발자 센터](https://developers.naver.com/apps/#/register) (선택사항)

### 설치 방법

1. **프로젝트 다운로드**
   ```bash
   # 이 폴더를 웹 서버에 업로드하거나 로컬에서 실행
   ```

2. **로컬 실행**
   ```bash
   # Python이 설치되어 있다면
   python -m http.server 8000
   
   # 또는 Node.js가 설치되어 있다면
   npx http-server
   ```

3. **브라우저에서 열기**
   ```
   http://localhost:8000
   ```

### API 키 설정

1. 애플리케이션을 처음 실행하면 API 키 입력 프롬프트가 표시됩니다
2. 또는 각 기능을 처음 사용할 때 API 키를 입력할 수 있습니다
3. API 키는 브라우저의 로컬 스토리지에 안전하게 저장됩니다

## 📖 사용 방법

### 1. 주제 발굴
1. 상단 네비게이션에서 "주제 발굴" 탭 선택
2. 5가지 모델 중 하나를 선택
3. 필요한 정보 입력 (카테고리, 키워드, 메모 등)
4. "주제 생성" 버튼 클릭
5. 생성된 주제 중 하나를 선택하여 포스트 생성으로 전달

### 2. 키워드 분석
1. "키워드 분석" 탭 선택
2. 분석할 키워드 입력
3. 경쟁도, 검색량, SEO 전략 확인
4. "포스트 생성으로 전달" 버튼으로 다음 단계 진행

### 3. 포스트 생성
1. "포스트 생성" 탭 선택
2. 제목 입력 (주제 발굴이나 키워드 분석에서 전달받을 수 있음)
3. 추가 요청사항 입력 (선택사항)
4. 컬러 테마 선택
5. 이미지 및 인터랙티브 요소 옵션 선택
6. 글쓰기 스타일 선택 (친근형/전문가형)
7. "포스트 생성" 버튼 클릭
8. 생성된 HTML 코드 복사하여 블로그에 붙여넣기

### 4. 이미지 생성
1. "이미지 생성" 탭 선택
2. **이미지 생성** 서브탭:
   - 이미지 설명 입력
   - 스타일 및 크기 선택
   - "이미지 생성" 버튼 클릭
   - 생성된 이미지 다운로드 또는 썸네일 제작에 사용
3. **썸네일 제작** 서브탭:
   - 배경 이미지 업로드 또는 생성된 이미지 사용
   - 텍스트 입력 (줄바꿈은 `/`로 구분)
   - 글꼴, 색상, 크기 조정
   - "썸네일 다운로드" 버튼 클릭

### 5. SNS 홍보
1. "소셜 홍보" 탭 선택
2. 블로그 글 제목과 요약 입력
3. "SNS 문구 생성" 버튼 클릭
4. 플랫폼별 문구 확인 및 복사

## ⚠️ 주의사항

### API 사용량
- Gemini API: 무료 티어는 분당 60회 요청 제한
- DALL-E 3: 유료 서비스 (이미지당 약 $0.04-$0.08)
- Naver API: 일일 25,000회 무료

### CORS 이슈
- 네이버 API는 CORS 정책으로 인해 프록시 서버가 필요할 수 있습니다
- 현재는 데모 데이터를 사용하도록 구현되어 있습니다

### 브라우저 호환성
- 최신 브라우저 사용을 권장합니다
- IE는 지원하지 않습니다

## 🔒 보안

- API 키는 브라우저의 로컬 스토리지에만 저장됩니다
- 서버로 전송되지 않습니다
- 정기적으로 API 키를 교체하는 것을 권장합니다
